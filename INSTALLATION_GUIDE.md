# Detailed Installation Guide for Noizzz

## Windows Installation Guide

1. **Download the Installer:**
   - Navigate to the [Releases](../../releases) page of this repository.
   - Look for the latest release (tagged as "Latest").
   - Under the "Assets" section, click on the file named `Noizzz-Setup-x.x.x.exe` to download it.

2. **Run the Installer:**
   - Once the download is complete, locate the `.exe` file in your Downloads folder.
   - Double-click the file to start the installation process.
   - *Note:* If Windows SmartScreen prompts "Windows protected your PC", click on **More info** and then select **Run anyway**. (This happens because the app is a new release and hasn't gathered a long reputation history yet).

3. **Follow the Setup Wizard:**
   - Follow the on-screen instructions. You can customize the installation directory if desired.
   - Wait for the installation to finish and click "Finish".
   - You can now launch Noizzz from your Desktop or Start Menu.

## Android Installation Guide

1. **Download the APK:**
   - Open your Android browser and go to the [Releases](../../releases) page of this repository.
   - Find the latest release and expand the "Assets" section.
   - Tap on `Noizzz-x.x.x.apk` to download the file.

2. **Enable Installation from Unknown Sources (If needed):**
   - If this is your first time installing an APK directly, your phone will block the installation for security reasons.
   - A prompt will appear saying "For your security, your phone is not allowed to install unknown apps from this source."
   - Tap **Settings** on the prompt, and toggle **Allow from this source** to ON.
   - Press the back button to return to the installation screen.

3. **Install the App:**
   - Tap **Install**.
   - Once installed, tap **Open** to launch Noizzz.
   - The app may request permissions for storage or media access to process your files locally. Please grant these permissions for the app to function properly.
