# Noizzz

Welcome to the official release repository for **Noizzz**! 
Noizzz is a powerful offline multimedia tool built for advanced audio and video processing, featuring noise reduction and source separation—all running locally on your device for complete privacy.

This repository is dedicated solely to distributing compiled releases (executables for Windows and APKs for Android). **No source code is hosted here.**

## 🚀 Features
- **Offline Processing:** No cloud uploads required. Your media stays on your device.
- **Noise Reduction:** Powered by advanced AI models to efficiently clean up your audio.
- **Source Separation:** Extract vocals, drums, bass, and other elements locally.
- **Media Conversion:** Robust media processing powered by FFmpeg.
- **Cross-Platform:** Available for Windows (.exe) and Android (.apk).

## 📥 Downloads
You can download the latest version of Noizzz from the [Releases page](../../releases).

### Available Formats
- **Windows:** `Noizzz-Setup-x.x.x.exe`
- **Android:** `Noizzz-x.x.x.apk`

## 🛠 Installation

### Windows (.exe)
1. Go to the [Releases](../../releases) section.
2. Download the latest `Noizzz-Setup-x.x.x.exe`.
3. Double-click the downloaded file and follow the setup wizard instructions.

### Android (.apk)
1. Go to the [Releases](../../releases) section from your Android device.
2. Download the latest `Noizzz-x.x.x.apk`.
3. Locate the APK file in your file manager and tap to install.
   *(Note: You may need to enable "Install unknown apps" from your browser or file manager settings).*
